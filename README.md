# [지능형 직업병 판단 시스템]

[지능형 직업병 판단 시스템]
KWCS 데이터를 활용해 직업군 별 직업병 예측시스템을 개발과 활용성 증대를 위해 접근성이 용이한 웹(WEB) 서비스를 제공하는 것을 목표로 한다.
